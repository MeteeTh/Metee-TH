This Capstone Project is Part of the Information & Knowledge Organization Modules (IKO)

Developed a website as a character database for Marvel movies using WordPress. The site showcases information about characters in the Marvel Cinematic Universe, including: Character details, Actor names, First appearances, Related movies and appearances.
