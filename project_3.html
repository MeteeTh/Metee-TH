ANDY Coffee & Friends POS System Database Development

This Capstone Project is Part of the Data Analytics Modules (DA)

Topic: Database Management and Data Warehousing

Designed and developed a beverage menu database system and a web application for managing beverage sales. The system allows for managing menu data (add/edit/delete) and generates daily and monthly revenue reports, as well as reports on best-selling items. The project involved the complete process from User Requirements, ER-Diagram, ER to Relations, Normalization, Input/Output Design, Database Design, Database Management with phpMyAdmin, UX/UI Design, to Development.

Capstone Project นี้เป็นส่วนหนึ่งของโมดูล การวิเคราะห์ข้อมูล (DA)

ในหัวข้อ: การจัดการฐานข้อมูลและคลังเก็บข้อมูล

ออกแบบและพัฒนาระบบฐานข้อมูลเมนูเครื่องดื่มของร้าน และ Web Application สําหรับการขายเมนูเครื่องดื่ม โดยให้ระบบสามารถจัดการข้อมูลเมนู (เพิ่ม/แก้ไข/ลบ) ได้ และมีระบบออกรายงานรายได้ประจําวัน, ประจําเดือน และเมนูขายดีได้ โดยดำเนินการตั้งแต่ User Requirements, ER-Diagram , ER to Relations, Normalization, Input Output Design, Database Design, Database Management โดยใช้ phpMyAdmin, UX/UI Design, Development

System results

The program features a sales window displaying available products, including menu names and prices. Users can add, remove, or clear order items, and the system displays the quantity of items, total payment amount, and change due. Payment options include both Cash and QR payment, with the ability to generate and display a QR code for scanning. Receipts can also be printed after orders. The admin window allows management of the menu, including adding, removing, and editing items. It also provides reports on daily sales, monthly sales, and best-selling items.

ฟังก์ชันของโปรแกรมประกอบด้วยหน้าต่างการขายที่แสดงข้อมูลสินค้าที่มี ชื่อเมนู และราคาขาย สามารถทําการเพิ่ม/ลบ/ล้างรายการการสั่งซื้อได้และแสดงจํานวนสินค้า, ยอดรวมที่ต้องชําระ และเงินทอน โดยสามารถ เลือกช่องทางการชําระเงินได้ทั้ง Cash และ QR payment และสามารถแสดง QR Code สําหรับการชําระแบบ สแกน QR Code สามารถพิมพ์ใบเสร็จการสั่งซื้อได้ ส่วนในหน้าต่างแอดมินจะแสดงรายการเมนูที่สามารถทําการเพิ่ม/ ลบ/แก้ไขเมนูได้และมีรายงานยอดขายประจําวัน, ประจําเดือน และเมนูขายดี
